Check version of iOS | 检查 iOS 版本

The iOS version should below iOS 9. It works on first gen iPad with iOS 5.1.1.

iOS 版本应该低于 iOS 9。已明确它可用在 iOS 5.1.1 的第一代 iPad 。

Check free space | 检查存储空间

Make sure you have at least 3GB free space on your device.

确保您的设备上至少有 3GB 可用空间。

Add QMole Linux repo to Cydia | 添加 QMole Linux 源到 Cydia

Install the IPG Certificates by opening http://cydia.invoxiplaygames.uk/certificates/ on your device and click on ISRG Root X1 CA (Let's Encrypt).

在设备上打开 http://cydia.invoxiplaygames.uk/certificates/ 后点击 ISRG Root X1 CA (Let's Encrypt) 以安装 IPG 证书。

Older versions of iPhone OS aren't new enough to have modern certificate authorities trusted, meaning only sites using older certificate authorities are able to have a secure connection.

旧版本的 iOS 系统无法让现代证书信任，若不安装新认证证书，就只能安全连接使用旧证书的网站。

Open this web page on your device and click on . Or just add http://qmole.pj568.eu.org in Cydia.

在设备上打开此网页后点击 。或直接在 Cydia 添加：http://qmole.pj568.eu.org。

Install QMole Usr Local Switch | 安装 QMole Usr Local Switch

Search and install QMole Usr Local Switch in Cydia.

在 Cydia 搜索并安装 QMole Usr Local Switch 。

The package makes a symlink from /usr/local to the user storage in order to gain space for QMole to install.

该软件包从 /usr/local 创建符号链接到用户存储，使 QMole 有足够的存储空间以安装。

Install QMole Desktop | 安装 QMole Desktop

Search and install QMole Desktop in Cydia.

在 Cydia 搜索并安装 QMole Desktop 。

The package enables you to access QMole Desktop environment.

该软件包使您能够访问 QMole 桌面环境。

Set time | 设置时间

Set your device's clock to a date before 31 August, 2016.

将设备的事件设置到任意于 2016 年 8 月 31 日 之前的日期。

For unknown reasons, QMole has a built in expiration date which will disable the "I Agree" / "Start" button if the date is old enough.

由于未知的原因，QMole 有一个内置的到期日期，如果日期到了，它将禁用“I Agree”或“Start”按钮。

Code view: #L454 #L460

Fixing Issues | 修复问题

Launch and set up the app | 启动并配置应用程序

Launch the QMole app on your homescreen.

在主屏幕上启动 QMole 应用程序。

Read the terms and conditions, then press "I Agree".

阅读条款和条件，然后点击“I Agree”。

Press "Start".

点击“Start”。
